# VTubelatro

![VTubelatro](images/VTUBELATRO.png)
*Art by DakiVT*

A [Balatro](https://store.steampowered.com/app/2379780/Balatro/) mod that adds new content based on VTubers. This mod is **not** meant to be balanced. It's meant to be silly and fun, so breaking a run with these Jokers is not a bug, it's a feature.

## Current Content

| Type                  | Count |
|-----------------------|-------|
| Jokers                | 48    |
| Seals                 | 3     |
| Consumables (Tarots)  | 3     |
| Tags                  | 6     |
| Editions              | 3     |
| Decks                 | 3     |
